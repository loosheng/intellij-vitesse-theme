- HTML Custom Tag

### Added
- add support for `html` and `xml`
- add support for `punctuation`
- add support for `operator`

- add support for `SearchEverywhereTab`

- optimize `Button`
- optimize `ProgressBar`
- optimize `focusColor`
- optimize dark `selectionColor`
